# Retail Sales Exploratory Data Analysis (EDA)

## Project Overview
Performed exploratory data analysis on retail sales dataset to uncover trends and business insights.

## Key Analysis
- Total Revenue Calculation
- Year-wise Sales Trend
- Product Category Performance
- Gender-wise Purchasing Analysis

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Identified top revenue-generating product categories
- Analyzed customer demographics
- Observed sales trends over time
